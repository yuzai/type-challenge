## 题目描述

实现高级 util 类型`OptionalKeys<T>`，该类型将 T 中所有可选属性的键合并为一个联合类型。


## 分析

之前已经讨论过获取可选属性，那么这题只需要把属性值取出来即可。送分题啦。

## 题解

```ts
type OptionalKeys<T> = keyof {
  [P in keyof T as T[P] extends Required<T>[P] ? never : P]: any
}
```
